Orientation – Spatial Awareness Test

Overview

This assessment measures your ability to interpret aircraft instruments and determine spatial orientation. You will be shown instrument panels and must select the correct aircraft orientation from four options.

Test Details

  • Time Limit: 7 minutes 30 seconds
  • Answer Format: Type the correct option number (1–4)
  • Navigation: Sequential (no going back)
  • Scoring: Percentage score shown at the end

Instructions

  • Type 1–4 and press Continue for each question.
  • On the last question, click Submit to finish the test.
  • The test will also end if time runs out.
